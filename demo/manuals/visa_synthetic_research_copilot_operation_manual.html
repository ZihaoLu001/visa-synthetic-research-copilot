Positioning

This is an early-stage hypothesis screening tool. It helps consultants identify promising propositions and survey design risks before commissioning real customer research.

Four-step operation

Upload

Attach a PDF survey or use the sample public survey.

Review

Check extracted survey text and edit questions if needed.

Configure

Set target market and adjust card propositions.

Run

Generate insights, persona responses and validation evidence.

Recommended reviewer setting

Use Model provider = watsonx for the final real-model proof once credentials are configured. Keep mock only as a fallback for rehearsal or quota issues. Use respondents = 96 and consistency runs = 2 for the full partner walkthrough.
