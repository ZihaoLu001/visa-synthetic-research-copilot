Scenario: Early-stage Swiss card value proposition concept test.

1. How appealing is this card concept overall?
2. How likely would you be to adopt this card if it were offered by your primary bank?
3. Which benefit would make you switch first? Options: grocery cashback; travel insurance; FX fee waiver; purchase protection; mobile wallet onboarding; none
4. Rank the following features: FX waiver / grocery cashback / lounge vouchers / fraud alerts / subscription controls

6. Would this card replace your current main card or be an additional card? Options: replace current main card; additional secondary card; would not use; unsure
7. What is missing or unclear in the proposition?
8. What main barrier would prevent you from using this card? Options: annual fee; privacy concern; setup complexity; weak rewards; no clear need; low trust
